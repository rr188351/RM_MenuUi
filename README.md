# RM_MenuUI Plugin

A reusable Unreal Engine plugin built on **Common UI** for scalable menu systems.

## ⚙️ Prerequisites
Before pasting the plugin into your project's `Plugins/` folder, make sure you:

1. **Enable Common UI** in Project Settings.
2. **Enable Enhanced Input** both as a plugin and in Project Settings.
3. Remember: **Localization of language** works properly only in **Standalone mode**, not in PIE (Play In Editor).

---

## 🎮 Features
- **Menu System** → Modular, reusable menus powered by Common UI.
- **Graphics Settings** → Resolution, quality presets, and display options.
- **Audio Settings** → Volume sliders, mute toggles, and sound profiles.
- **Display Controls** → Fullscreen/windowed modes, brightness, and scaling.
- **Input Controls** → Enhanced Input mapping with profiles for keyboard, controller, and touch.
- **Save/Load System** → Game settings persist across sessions.
- **Credits Screen** → Cinematic scrolling credits with audio integration.
- **Continue/Resume Flow** → Seamless game resume from last save.
- **Asynchronous Operations** → Smooth UI updates without blocking gameplay.

---

## 🚀 Installation
1. Copy the `RM_MenuUI` folder into your project's `Plugins/` directory.
2. Restart Unreal Engine.
3. Enable the plugin from `Edit → Plugins`.
